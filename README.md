Welcome to the Bioinformatics Toolkit! This is a modular, interactive Python script designed to assist biologists and researchers in performing standard DNA sequence analyses quickly and accurately.

Important Note: This script (Version 1.0) uses the standard codon tables and motif definitions applicable to Human/Eukaryotic DNA. Please do not use this for prokaryotic sequence analysis without modifying the dictionaries.

How to Use
Run the bioinformatics_toolkit.py file in your terminal or command prompt. You will be greeted with an interactive main menu where you can type the numbers of the specific analyses you wish to run (e.g., 1,3,5 or all).

If you are processing Multi-Sequence FASTA files (Step 8), ensure the .fasta file is in the same directory as the Python script, or provide the absolute path when prompted.
